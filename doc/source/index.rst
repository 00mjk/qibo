================================
Welcome to Qibo's documentation!
================================

Qibo is an open-source high-level API, written in Python and capable
of running a quantum algorithms on top of different quantum computers
and simulators. It provides a standard interface, allows the
implementation and extension of new algorithms and eliminates the
requirement of adjust code in order to run option different backends
and architectures.
